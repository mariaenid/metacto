# Architectural Decisions

One file per decision, named `NN-<kebab-slug>.md` with `NN` zero-padded (e.g. `01-clean-architecture-layers.md`).

## Template
```
# NN — Title

## Context
Why this decision was needed.

## Decision
What was decided.

## Consequences
Trade-offs accepted and follow-up implications.
```

Keep entries short. The goal is durable rationale, not exhaustive analysis.
